# Overview

This is an example of node application for research and development. It is a simple application to demonstrate how to create a node application that prints "Hello, World!" to the console. The application starts simple and then new features will be added to create a module application for show case purposes.

# Development Environment

The application uses Visual Studio Code with Node.js version 22.22.0 or greater and NPM for development.
