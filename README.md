# Overview

This is an example of node application for research and development. It is a simple application to
demonstrate how to create a node application that prints "Hello, World!" to the console. The application
starts simple and then new features will be added to create a module application for show case purposes.

# Development Environment

This application uses nodejs version 22.22.0 and npm 10.9.4.

```bash
npm run dev
# or
node src/index.js
```

Create and start a the PostgreSQL database container:

```bash
docker-compose up -d
# or
podman-compose up -d
```
